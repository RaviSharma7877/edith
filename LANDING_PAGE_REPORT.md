## Problem Section

Section headline:

> Most business tools separate the workflow from the books.

Problem points:

- Sales teams create promises that accounting has to reconstruct.
- Invoices, bills, payments, and journals drift across different tools.
- Reconciliation becomes a monthly cleanup instead of a daily control.
- Reports are fast to open but hard to trust when source data is scattered.
- AI assistants often lack financial context and approval boundaries.

Closing line:

> Edith brings the workflow and the books into the same controlled system.

Section headline:

> AI that knows the workspace, but respects the ledger.

> Edith's AI copilot helps explain, summarize, draft, and suggest. It does not post journals, file returns, close periods, or reverse transactions without deterministic validation and human approval.

- Explain report variance

- Map import columns
- Summarize client documents
- Draft proposal sections
- Create task suggestions
- Answer questions using selected page context

AI context examples:

- This report
- Selected customer
- Open invoices
- Reconciliation batch
- Current proposal
- Uploaded document

Guardrail copy:

> AI can suggest, summarize, draft, and explain. Posting, filing, closing, and reversing stay under accounting controls.
